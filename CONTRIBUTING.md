## Contributing

Thanks for your interest in the FIFA World Cup 2026 Tracker! Here are a few ways you can help:

### Quick Start

If you just want to test the live demo, visit **[project516.github.io/wc2026-tracker](https://project516.github.io/wc2026-tracker/)** and you can interact with the standings, matches, and scorers.

### For Developers

#### Local Development

This is a static site — no build steps are needed.

1. Clone the repo:
   ```sh
   git clone https://github.com/Project516/wc2026-tracker.git
   cd wc2026-tracker
   ```

2. Open `index.html` in any modern browser to see the live preview.

#### Feature Ideas

This project uses public ESPN API endpoints, so most content updates automatically. If you notice missing data, broken formatting, or a feature you'd like to see, please [open an issue](https://github.com/Project516/wc2026-tracker/issues/new).

### Reporting Bugs / Suggesting Features

When opening an issue, please include:

- Steps to reproduce (if applicable)
- What you expected to see
- What you actually saw
- Browser and device details (optional but helpful)

### Pull Requests

We follow the GitHub Flow workflow:

1. Fork the repo and create a feature branch.
2. Make your changes (editing HTML, CSS, or JS, or improving documentation).
3. Add a `.gitignore` if missing and remove any build artifacts (see `.gitignore` for what's ignored).
4. Commit your changes with a clear, descriptive message.
5. Push and open a pull request against `Project516/wc2026-tracker:master`.

Please keep PRs focused and minimal — one change per PR if possible.

### Code Style

Because this is vanilla HTML/CSS/JS, we rely on:

- Visually clean markup and consistent indentation
- No extra console noise (no `console.log` should be committed)
- Semantic HTML where appropriate

If you encounter formatting or lint issues, feel free to fix them as part of your changes.

Thanks for helping make this tracker even better!
